# Catalyst Analytics R v1.3.0

## Optimization and Policy Pathway Design

This release adds constrained single- and multi-objective optimization, grid and reproducible random search, feasible-policy regions, Pareto frontiers, target-seeking scenarios, cost-effectiveness, marginal-abatement curves, adaptive stages, evidence triggers, robust pathway regret, workspace libraries, portable publication bundles, and WordPress companion v2.3.0.

All outputs preserve a human-review boundary. Recommendations do not authorize or execute policy.
