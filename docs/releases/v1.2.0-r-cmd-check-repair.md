# Catalyst Analytics R v1.2.0 R CMD check repair

This same-version repair normalizes regional portfolio `price_year` values when JSON contains `null` or omits the field. The importer restores a named `NA_integer_` value before validation, and construction and validation use the same nullable scalar contract.

Regression coverage includes explicit JSON null, omitted fields, direct `NULL` construction, non-scalar input, and out-of-range years.
