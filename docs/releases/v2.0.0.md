# Catalyst Analytics R v2.0.0

## Connected Sustainability Analytics and Decision Platform

This major release adds the connected analytical graph, cross-product lineage, federated registries, evidence-backed decision and publication records, governed workflow orchestration, connected API v2, and portable connected-platform publication bundles.

Stable v1 analytical contracts remain supported and are referenced by the v2 platform rather than rewritten.

## Connected-platform fingerprint repair

Canonical JSON exports now record the platform's semantic fingerprint in `metadata.serialized_platform_fingerprint`. Imports preserve that identity across JSON type normalization while tracking the imported semantic state. Any subsequent platform mutation invalidates the restored identity and produces a new fingerprint.
