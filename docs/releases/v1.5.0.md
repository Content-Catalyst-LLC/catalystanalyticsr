# Catalyst Analytics R v1.5.0

## Public API and Sustainable Catalyst Handoffs

This release adds a stable v1 endpoint manifest, request and response envelopes, read-only in-process dispatch, six governed first-party platform handoffs, portable handoff bundles, workspace storage, schemas, fixtures, tests, migration guidance, and WordPress companion v2.5.0.

The release does not include a network server, credential storage, or autonomous cross-product execution. Host products remain responsible for transport, authentication, authorization, rate limiting, retention, and human approval.
