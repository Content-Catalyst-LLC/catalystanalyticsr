# Catalyst Analytics R v0.3.0

## Comparative Scenario Engine

Released: July 2026

### Added

- `run_scenarios()` governed batch execution
- `catalyst_scenario_set` print method
- `compare_scenarios()` comparative analysis
- `catalyst_comparison` print and plot methods
- `scenario_deltas()`
- `scenario_rankings()`
- `scenario_scorecard()`
- `pareto_diagnostics()`
- Direction-aware metric comparison
- Automatic and explicit baseline resolution
- Target and threshold rules
- Trade-off classification
- Pairwise dominance and Pareto layers
- Overlaid trajectory, terminal, delta, and trade-off plots
- `export_scenario_comparison()` reproducible bundles
- Comparative input and export schemas
- Comparative fixtures and release tests
- WordPress demo v1.2.0 with baseline-versus-policy analysis

### Changed

- Repository and R package version advanced to 0.3.0.
- WordPress demo version advanced to 1.2.0.
- Browser export contract advanced to 1.2.0.
- Public demo wording now describes a real comparison rather than a single scenario.
- Release contract now requires comparative schemas, fixtures, R APIs, tests, and browser behavior.

### Compatibility

- Canonical scenario schema remains 1.0.0.
- Model registry contract remains 1.0.0.
- KH-NC-PA model remains 1.0.0.
- Legacy single-scenario browser input remains available for migration and compatibility testing.

### Boundaries

Comparative results are exploratory decision support. Dominance, rankings, targets, and Pareto status do not constitute autonomous recommendations or professional advice.
