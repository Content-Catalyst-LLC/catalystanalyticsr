# Catalyst Analytics R v0.4.0

## Uncertainty, Sensitivity, and Stress Testing

Released as the fourth governed analytical foundation release.

### Added

- Versioned uncertainty specifications embedded in canonical scenarios.
- Monte Carlo and Latin hypercube sampling with reproducible seeds.
- Fixed, uniform, normal, lognormal, triangular, beta, and discrete distributions.
- Scenario ensembles with visible failure accounting.
- P2.5, P10, median, P90, and P97.5 intervals.
- Threshold-crossing probabilities.
- Spearman and Pearson global sensitivity diagnostics.
- One-at-a-time local sensitivity and elasticity analysis.
- Named structural stress cases and shock application modes.
- Uncertainty and stress-test plots and export bundles.
- WordPress demo v1.3.0 with browser uncertainty bands and probability summaries.
