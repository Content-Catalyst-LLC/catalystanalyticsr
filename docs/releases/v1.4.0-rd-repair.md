# Catalyst Analytics R v1.4.0 Rd Usage Repair

This same-version repair corrects generated documentation for the econometrics and workspace policy-evaluation APIs. The original files embedded literal `\n` text inside `\usage{}` sections, so R could not parse the documented calls as syntactically valid R code.

The repair replaces those tokens with real line breaks, preserves all function aliases and arguments, and adds release-contract tests that reject escaped newlines or unbalanced usage signatures. No analytical calculations, schemas, or WordPress behavior changed.
