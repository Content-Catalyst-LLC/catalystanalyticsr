# Catalyst Analytics R v0.9.0

**Release:** Reproducible Projects and Analytical Publication

This release adds governed project structures, complete run histories, software-environment capture, stable input/output hashing, project snapshots, review records, portable publication bundles, and Decision Studio and Knowledge Library handoffs. WordPress companion v1.8.0 maps the browser experience to the project and publication contracts without claiming R execution or numerical parity.


## R CMD check repair

The repaired release makes publication file counts explicit, declares that manifest integrity covers all generated bundle files except the manifest itself, and qualifies `utils::capture.output()` so environment capture is namespace-clean.
