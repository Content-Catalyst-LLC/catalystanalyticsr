# Review Checklist

Use this checklist before publishing or relying on a Catalyst Analytics R output.

- [ ] Scenario name is clear.
- [ ] Time horizon is stated.
- [ ] Initial conditions are visible.
- [ ] Policy controls are documented.
- [ ] Emissions budget is stated.
- [ ] Indicator definitions are clear.
- [ ] Export is reproducible.
- [ ] Interpretation notes identify uncertainty.
- [ ] Output does not claim certainty, certification, or compliance.
- [ ] Human review has occurred.
